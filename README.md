# Tip-Calculator


## How to run it locally

You do not need to install complex tools or use the terminal. It is very easy to run.

**Option 1: Open in your browser (Fastest)**
1. Clone this repository to your computer or click `Code -> Download ZIP` on GitHub.
2. Open the downloaded folder.
3. Double-click the `index.html` file. It will open in your web browser (like Google Chrome or Safari).

**Option 2: Use VS Code (For coding)**
If you use the VS Code program, you can use the "Live Server" tool:
1. Open the folder in VS Code.
2. Right-click the `index.html` file.
3. Click "Open with Live Server". The app will open in your browser automatically.
